# 📘Data Dictionary – student_insights (Gold Table)

| Column                     | Description                           |
| -------------------------- | ------------------------------------- |
| id_student                 | Unique student ID                     |
| code_module                | Course module                         |
| code_presentation          | Course presentation/session           |
| gender                     | Student gender                        |
| region                     | Student region                        |
| highest_education          | Student education level               |
| age_band                   | Student age group                     |
| studied_credits            | Credits taken by student              |
| num_of_prev_attempts       | Previous attempts                     |
| disability                 | Disability indicator                  |
| final_result               | Final course result                   |
| total_clicks               | Total VLE clicks (student engagement) |
| avg_score                  | Average assessment score              |
| weighted_score             | Weighted assessment score             |
| date_registration          | Registration date                     |
| date_unregistration        | Unregistration date                   |
| module_presentation_length | Course duration                       |
